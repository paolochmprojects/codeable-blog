Como Usar el Elemento HTML <audio>

El elemento HTML <audio> se utiliza para incrustar contenido de audio en un documento HTML. Este elemento puede utilizarse para reproducir música, podcasts, audiolibros, archivos de sonido y otros tipos de contenido de audio.

No olvides, entre la etiqueta de apertura y cierre es recomendable colocar un aviso en texto, este texto se mostrará en caso el navegador del usuario no sea compatible o no reconozca dicho elemento de audio.

Sintaxis del Elemento <audio>

La sintaxis básica del elemento <audio> es la siguiente:
<audio> </audio>

Atributos del Elemento <audio>

Aqui te encontraras con una lista de todos los atributos del elemento <audio>, junto con una breve descripción de cada uno:

  1. `src`: Especifica la ubicación del archivo de audio o la URL del archivo de audio. Este es el atributo más importante del elemento <audio>, ya que determina qué archivo de audio se reproducirá.
  2. controls: Muestra los controles de reproducción de audio, como el botón de reproducción, el botón de pausa, el botón de avance rápido y el botón de retroceso.
  3. loop: Hace que el audio se reproduzca en bucle.
  4. muted: Silencia el audio.
  5. preload: Especifica si el audio se cargará de forma automática o no.
  6. autoplay: Hace que el audio comience a reproducirse automáticamente cuando se carga la página.
  7. controlslist: Especifica qué controles de reproducción de audio se mostrarán.
  8. mediagroup: Agrupa varios elementos <audio> para que se reproduzcan de forma sincrónica.
  9. preloadend: Especifica cuánto tiempo de audio se ha cargado en el navegador.
  10. controlslistend: Especifica qué controles de reproducción de audio se mostrarán cuando se haya cargado la mayor parte del audio.

Además de estos atributos, el elemento <audio> también puede heredar todos los atributos globales de HTML.

Aquí hay algunos ejemplos de cómo se pueden utilizar estos atributos:


  • Para reproducir un archivo de audio llamado "music.mp3" en bucle, sin controles y sin silenciar: 

    • <audio src="music.mp3" loop muted controls/>
  • Para reproducir un archivo de audio llamado "podcast.ogg" de forma automática: 

    • <audio src="podcast.ogg" autoplay/>
  • Para reproducir varios archivos de audio de forma sincrónica, agrupados en un grupo llamado "my-audio-group": 

    •       <audio src="audio1.mp3" controls>
       <audio src="audio2.mp3" controls>
       <audio src="audio3.mp3" controls>

Etiqueta <source>para insertar audio

Otra manera de insertar elementos de audio es utilizando la etiqueta <source>dentro de una <audio>,dentro de ella se especifica cual es el archivo mediante el atributo`src`y además se puede especificar el tipo de archivo utilizando el atributo `type.`Veamos un ejemplo acontinuacion. 

<audio controls>
  <source src="miaudio.mp3" type="audio/mp3">
  Este es un elemento de audio no soportado por 
  tu navegador prueba con otro
</audio>

Formatos de audio soportados

No todos los formatos de audio que existen se pueden utilizar para insertar en una página web, en esta ocasión conoceremos los formatos más recomendables.

  • MP3
  • OGG
  • WAV
  • AAC
  • OPUS
